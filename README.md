# Personal Website – Data Engineer Portfolio

Includes:
✅ Projects  
✅ Markdown Blog (using react-markdown)  
✅ Versioned Footer  
✅ Clean GitHub Pages deployment via `/docs` folder

## 🚀 Usage

```bash
npm install
npm run dev
```

Then deploy:

```bash
chmod +x deploy.sh
./deploy.sh
```

Ensure GitHub Pages is set to:
➡️ Branch: `main`  
➡️ Folder: `/docs`

Visit: https://liamnomad.github.io
